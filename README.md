##Description

Python program to operate on downloads data from yahoo finance and converts it into csv then in json and yaml.

## Required

- Python version between `3.6` and `3.9`
- `requests` library
    pip install request
    
- `csv` library
    pip install csv

- `json` library
    pip install json
    
- `Yaml` library
    pip install PyYAML
    
- `pandas` library
    pip install pandas

## Execution examples

Open a terminal with path "./portafoglio/" and type the following command:

- Windows:
  ```
  python main.py
  ```

- Linux:
  ```
  python3 main.py
  ```
